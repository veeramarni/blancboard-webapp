jodconverter
===============
Make sure to load sigar.jar when deploying
mvn install:install-file -DgroupId=org.hyperic.sigar -DartifactId=sigar -Dpackaging=jar -Dversion=1.6.5 -Dfile=C:\hyperic-sigar-1.6.5\sigar-bin\lib\sigar.jar
mvn install:install-file -DgroupId=org.apache.xmlgraphics -DartifactId=pdf-transcoder -Dversion=0.94 -Dpackaging=jar -Dfile=E:\IDE\ExternalJars\pdf-transcoder.jar 
mvn install:install-file -DgroupId=jpedal -DartifactId=jpedal -Dversion=4.6 -Dpackaging=jar  -Dfile=E:\IDE\ExternalJars\jpedal_lgpl.jar 
mvn install:install-file -DgroupId=com.artofsolving -DartifactId=jodconverter-core -Dversion=3.0-beta-4 -Dpackaging=jar  -Dfile=jodconverter-core-3.0-beta-4.jar

Jpedal
=============
To run JAI you need the jars on the classpath and the JVM flag -Dorg.jpedal.jai=true set.

Environment
===============
production - prod server
dev	       - demo server
local      - local server
test	   - test server (mercury)

Open office
=====================
open office settings are taken from emyed-properties file, make sure the following are correct and valid.

openoffice.listener.port=8100
openoffice.home.path=/opt/openoffice4
